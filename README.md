# gatk-sv-annotation-buffalo

Structural variant (SV) detection, annotation, and donor-clone comparison pipeline for Bubalus bubalis (river buffalo) genome using Oxford Nanopore Technology (ONT) long-read sequencing and GATK.

---

## Overview

This pipeline processes raw VCF files generated by cuteSV from ONT long-read whole genome sequencing data. It performs :

- Quality filtering of structural variants
- Out-of-bound chromosome detection
- Gene-level annotation
- Donor–clone comparative analysis

All analyses are performed using the buffalo reference genome **GCF_019923935.1 (NDDB_SH_1)** with NCBI RefSeq gene annotation.

---

## Pipeline Workflow

The analysis pipeline consists of four sequential steps:

### 1. **VCF Preprocessing**
   - **Script:** `01_vcf_preprocessing.sh`
   - Filters raw SV calls using GATK VariantFiltration.

### 2. **Chromosome QC and Out-of-Bound Detection**
   **- Script:** `02_oob_detection_qc.R`
   - Identifies structural variants outside expected genomic boundaries.

### 3. **SV Annotation**
   **- Script:** `03_sv_annotation.R`
   - Annotates SVs with gene names, genomic features, and biotypes.

### 4. **Donor–Clone Comparison**
   **- Script:** `04_sv_comparison.R`
   - Performs comparative structural variant analysis between donor and clone samples.

## Input Files
* Raw VCF files from cuteSV (ONT long-read data)
* Reference genome: GCF_019923935.1 (Bubalus bubalis, NDDB_SH_1)
* Gene annotation: genes.gtf (NCBI RefSeq)

## Genome Information
The pipeline was developed using the following genome assembly:

- Species: *Bubalus bubalis* (River Buffalo)
- Assembly: GCF_019923935.1 (NDDB_SH_1)
- Chromosomes: 25 nuclear chromosomes (chr1–chr24 + chrX)
- Mitochondrial genome: Not included
- Gene annotation: NCBI RefSeq GTF
